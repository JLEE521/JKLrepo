How to install and use VS codes live code extension

Install | use | Video | Image

Installation Instructions

  1. In Vs code, click the extensions icon on the left toolbar
  2. In the search bar that appears type: live server.
  3. select the live server extesion (should be top choice).
  4. on the page that appears click the green install button

Usage instructions

  • VS code must be open to a folder to use live server
  • In vs codes explorer, right click on an html file. from the context menu that appears select open live server
